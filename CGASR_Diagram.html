4. Detailed Data Flow: "How does WACC affect the valuation?"

sequenceDiagram
    participant U as User
    participant FE as Frontend
    participant API as /ask Endpoint
    participant R as CGASR Retriever
    participant IDX as CGASRIndex
    participant LLM as LLM (Groq)

    U->>FE: "How does WACC affect the valuation?"
    FE->>API: POST /ask {query, mode: "ask"}
    API->>R: retrieve(query, uuid, k=15)
    
    Note over R: Step 1: Direct lookup → NO (reasoning query)
    
    R->>R: embed(query) → q ∈ ℝ⁷⁶⁸
    R->>R: Router → CGASR (cross-sheet keywords)
    
    R->>IDX: cell_relevances = E · q
    Note over IDX: Assumptions!B13 (WACC) scores highest
    
    R->>IDX: spectral energy → coarse scale s=32 dominates
    Note over IDX: Query needs global propagation
    
    R->>IDX: PersonalizedPageRank(teleport=relevances)
    Note over IDX: π_q concentrates on WACC cell + downstream
    
    R->>IDX: wavelet_propagate(π_q, scales=[0.5,2,8,32])
    Note over IDX: s=32 wavelet propagates WACC importance
through Assumptions→P&L→CashFlow→DCF chain
    
    R->>R: score_chunks() with role weights
    Note over R: DCF chunk (TOTAL cells) scores highest
Assumptions chunk scores 2nd
P&L chunk scores 3rd
    
    R->>R: uncertainty estimation
    Note over R: σ_DCF = 0.15 (confident)
σ_Assumptions = 0.10 (very confident)
    
    R->>R: MMR ranking → top 15 chunks
    R-->>API: chunks + scores + σ
    
    API->>API: build_context(chunks, max=12000)
    API->>LLM: system + context + cell_index + query
    LLM-->>API: "WACC (10%) in Assumptions!B13 affects..."
    API-->>FE: SSE stream answer
    FE-->>U: Answer with confidence indicator
